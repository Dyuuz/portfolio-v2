Project Overview

MindyBrows is a production microservices platform built to run the day-to-day operations of a real makeup education and services business — from public-facing marketing pages to class registration, payment verification, and automated student communication across WhatsApp and Telegram.

The platform is built around automation, reliability, and reducing manual admin work. Students browse classes and services, register for a course, and upload proof of bank transfer payment directly on the site. Once an admin approves or rejects a registration, the system automatically fires off WhatsApp confirmations, sends a personalized Telegram invite link to a private class channel, and verifies channel membership all without manual follow-up.

A background system keeps everything running smoothly by retrying failed WhatsApp messages, sending verification reminders, and re-delivering invites when needed. Admins are instantly notified of service issues by email, and students can check their registration status anytime using their transaction ID.

Use cases include: makeup class registration and payment verification, automated multi-channel student onboarding (WhatsApp + Telegram), admin-driven approval workflows, scheduled retry handling for failed notifications, and service health monitoring across distributed microservices.

Current status: The public website, class registration and payment flow, admin approval system, WhatsApp and Telegram microservices, and the full background operation/data pipeline are complete and live in production.

MindyBrows is best positioned as a real-world example of tying a central application together with independent messaging services to fully automate a service business's enrollment pipeline which is built to run unattended, with built-in recovery when any one piece fails.
