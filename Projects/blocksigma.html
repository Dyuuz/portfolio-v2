Project Overview

ALX Project Nexus is a full-stack ecommerce platform built to handle real-world product catalogues, order management, and payment processing at scale. The backend is engineered with Django and Django REST Framework, exposing a clean, versioned API consumed by a responsive frontend.

The system handles user authentication, product inventory, cart management, order lifecycle, and payment gateway integration — with structured logging, test coverage, and a CI/CD pipeline shipping to production reliably on every merge.
